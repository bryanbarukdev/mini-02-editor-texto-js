# Editor de Texto com Desfazer e Refazer

![Status](https://img.shields.io/badge/status-em%20desenvolvimento-orange)
![Licença](https://img.shields.io/badge/license-MIT-green)

## Descrição
Editor minimalista que implementa operações de desfazer/refazer com duas pilhas, simulando o comportamento básico de aplicativos de edição de texto.

## Tecnologias
- JavaScript
- HTML
- CSS

## Funcionalidades
- Adicionar palavra ao texto.
- Desfazer última ação.
- Refazer ação desfeita.
- Atualização visual da textarea em tempo real.

## Como executar
1. Clone o repositório.
2. Abra `index.html`.
3. Teste fluxos: adicionar -> desfazer -> refazer.

## Aprendizados
- Aplicação prática de pilhas (LIFO).
- Gerenciamento de estado com histórico de ações.
